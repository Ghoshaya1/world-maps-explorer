# World Maps Explorer

## Description
Auto-generated project by AI Agent Platform

## Tech Stack
- React 18\n- TypeScript\n- Vite\n- React Leaflet\n- Leaflet\n- Tailwind CSS\n- React Query\n- Zustand\n- REST Countries API\n- ESLint\n- Prettier

## Features
- Interactive world map with zoom and pan controls\n- Country selection and highlighting\n- Country information panel with population, capital, and flag\n- Search functionality to find countries\n- Multiple map layers (street, satellite, terrain)\n- Responsive design for mobile and desktop\n- Bookmarking favorite countries\n- Dark/light theme toggle

## Getting Started

### Prerequisites
Install dependencies as per tech stack above.

### Installation
```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install  # or yarn install, pip install -r requirements.txt, etc.
```

### Running
```bash
npm start  # or appropriate command for your stack
```
